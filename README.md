# LoonBot

> Requires Node LTS / NPM

> Run: npm install

> Create oauth key https://twitchapps.com/tmi/

> Create .env file
```
BOT_USERNAME=your_bot_name
CHANNEL_NAME=you_channel_name
OAUTH_TOKEN=your_oauth_key
```


> RUN: node loonbot.js


> current cmds for channel owner / bot only
> !bbywrath {username},
> !bbygirl {username},
> !bbyboy {username},
> !vip {username}
